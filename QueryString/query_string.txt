Query String -
	query string module is used to parse and format url query strings.
	

querystring.parse()
		- this method is used to parse URL query string into key value pair .
		
		For example, the query string 'foo=bar&abc=xyz&abc=123' is parsed into:

			{
			  foo: 'bar',
			  abc: ['xyz', '123']
			}

querystring.stringify()-
		-  this method is used to create/produce URL query string from a given object.
			
			querystring.stringify({ foo: 'bar', baz: ['qux', 'quux'], corge: '' });
			// Returns 'foo=bar&baz=qux&baz=quux&corge='

querystring.escape(str)-
		-  this method performs URL encoding of given with. querystring.stringify() uses this methos internally.
		- Not intended to use explecitly.

querystring.unescape(str)-
		- performs URL decoding of given encoded string. querystring.parse() uses this methos internally.
		- Not intended to use explecitly.
